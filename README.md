# news_app

NewsApp เป็นแอปพลิเคชั่นมือถือที่ใช้ Flutter ที่ให้ผู้ใช้สามารถติดตามข่าวสารล่าสุดจากแหล่งต่างๆ แอปใช้ API ข่าวสารเพื่อดึงบทความข่าวและแสดงในส่วนต่อประสานที่ใช้งานง่าย
